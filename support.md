# Support PocketCloud

PocketCloud is a free, open-source project built and maintained in my free time.

If you find it useful, here are some ways you can support the project:

---

## ⭐ Star the repo

The simplest way to support — just hit the star button at the top of the page. It helps others discover the project.

---

## 🐛 Report bugs

Found something broken? Open an issue on GitHub with:
- What happened
- What you expected to happen
- Your Android version and device model

---

## 💡 Suggest features

Have an idea? Open an issue with the label `enhancement` and describe what you'd like to see.

---

## 🔧 Contribute code

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Commit your changes
4. Open a Pull Request

All contributions are welcome, big or small.

---

## 📣 Spread the word

Share PocketCloud with friends, post about it, or recommend it to anyone who wants to stop paying for cloud storage.

---

Thank you for using PocketCloud ☁️
